# 어휘 시험 출제기 (Vocab Test Maker)

엑셀 파일로 영어 단어 시험지를 자동 생성하는 웹 앱입니다.

## 기능

- 엑셀 파일 (.xlsx) 업로드 → 자동 단원 인식
- 시험 범위(단원)와 출제 단어 개수 자유 선택
- 영→한 / 한→영 출제 유형 전환
- 클래스명: 파일명 자동 추출
- 시험지명: 선택 단원 범위로 자동 생성
- 정답 보기 / 순서 섞기 / 인쇄 지원

## 엑셀 파일 형식

- A열: 단원번호 (숫자 또는 텍스트)
- B열: 영어 단어
- C열: 한글 뜻
- 헤더 행 없이 첫 행부터 데이터 입력

## 로컬 실행

```bash
npm install
npm run dev
```

## 배포 (Vercel)

GitHub에 push 후 Vercel에서 import만 하면 자동 배포됩니다.
