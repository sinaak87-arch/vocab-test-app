# 어휘 시험 출제기 (Vocab Test Maker)

구글 스프레드시트로 영어 단어 시험지를 만드는 웹 앱입니다.

## 사용법

1. **구글 스프레드시트 준비**
   - A열: 단원번호, B열: 영어 단어, C열: 한글 뜻
   - 헤더 행 없이 첫 행부터 바로 데이터 입력
   - [공유] → "링크가 있는 모든 사용자" → "뷰어"로 설정

2. **앱에서 시트 URL 입력 → [불러오기]** 클릭

3. **출제 옵션 선택**
   - 시험 범위 (단원)
   - 출제 단어 개수
   - 영→한 / 한→영 출제 유형

4. **[시험지 만들기]** → 미리보기 확인 → **[인쇄하기]**

## 로컬 실행

```bash
npm install
npm run dev
```

## 배포

이 프로젝트는 Vercel로 바로 배포 가능합니다. GitHub에 push 후 Vercel에서 import만 하면 됩니다.
